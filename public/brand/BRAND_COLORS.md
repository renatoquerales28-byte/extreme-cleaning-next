
# Extreme Cleaning Services - Brand Assets

## "Sentient" Color Palette
Use these precise codes to match the web application.

| Color Name | Hex Code | Purpose |
|------------|----------|---------|
| **Fuchsia** | `#FF00E5` | Primary Accents, Glows, "Commercial" |
| **Cyan** | `#00E0FF` | Residential Highlights, Gradients |
| **Emerald** | *Tailwind default* | Property Management / Trust |
| **Black** | `#000000` | Backgrounds, Strong Buttons |
| **White** | `#FFFFFF` | Text, Glass Effects |

## Storage Instructions
- **Logo**: Place your logo files here. Recommended name: `logo.png` or `logo.svg`.
- **PDFs**: You can upload your PDF guide here.

## Access
Once deployed, these files will be accessible at:
- `https://your-site.com/brand/logo.png`
- `https://your-site.com/brand/brand-guide.pdf`
